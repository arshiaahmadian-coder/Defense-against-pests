# Defense against pests

## About

A game developed with Unity and C#.

## Genre

Tower Defence / Strategy / Survival

## Features

* Farming and crop growth mechanics
* Wood production through farming
* Defensive building placement system
* Enemy insect waves attacking the farm
* Automated defensive structures
* Resource management and progression
* Base defense gameplay loop
* Multiple enemy and building types
* Object-Oriented architecture for game entities

## My Role

Solo Developer (Programmer / 3D Artist / Game Designer)

## Technologies

* Unity
* C#
* Belender
* Git
